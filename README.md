# quiz_curio

A simple quiz platform project.

## Features

- User authentication
- Create, edit, and delete quizzes
- Take quizzes and view results
- Basic admin panel

## Technologies Used

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/ArmanAmreliya/quiz_curio.git
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Start the server:
   ```
   npm start
   ```
